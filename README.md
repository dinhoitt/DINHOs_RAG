# DINHO_rag

광운대학교 대학원 수업 'LLMs 극복의 RAG 개발' 기말 프로젝트용 레파지토리입니다.

## 📢프로젝트 소개

논문 세미나 발표를 위한 지능형 요약 및 대본 생성 RAG

## ⚙개발 환경

- python `3.10.19`

## 📍 Vibe coding

- prompts.py 에 있는 System prompts는 온전히 챗지피티가 작성해준 프롬프트입니다.

- 이외의 모든 코드는 [교재 및 깃허브링크](https://github.com/teddylee777/langchain-kr) 를 참고하였습니다.

- 버전에 맞는 코드 변경, 프로젝트 구조에 맞는 프레임워크 설계, 프로젝트에 맞는 파라미터 변경 외에는 바이브 코딩은 없습니다.
